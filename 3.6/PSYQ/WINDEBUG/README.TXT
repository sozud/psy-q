
Abstract: psyq\windebug directory 11/18/96

Note: You will need two blank floppies.

The directory psyq\windebug contains the Psy-Q Windows Debugger.  Please copy the 
contents of "disk 1" to a blank disk. For example, 
if "a" is your floppy and "windebug"'s full path is "d:\psyq\windebug",
then
	
	copy d:\psyq\windebug\disk1\psxrel15.zip a:\

Remove the floppy. Insert another floppy.
Copy the contents of  "disk2" to another blank disk.  

	copy d:\psyq\windebug\disk2\psxrel15.zip a:\

You now have the two diskettes that we receive from Psy-Q.
Unzip the files in the disk and follow the onscreen instructions.
