\sample\sound\tuto\tuto1.c

  DMA 転送で送られた sin 波を、コントロールパッドの操作に
  よってさまざまな音程で発音するプログラムです。

	Lleft  ボタン (←ボタン):	低音
	Lup    ボタン (↑ボタン):
	Lright ボタン (→ボタン):
	Ldown  ボタン (↓ボタン):
	Rleft  ボタン (□ボタン):
	Rup    ボタン (△ボタン):
	Rright ボタン (○ボタン):
	Rdown  ボタン (×ボタン):	高音

    の順で音程が高くなっていきます。

	K ボタン (SELECT):	クローズ (演奏終了)

【プログラムの作成】

	> psymake tuto1

\sample\sound\tuto\tuto2.c

  ミュートのサンプルプログラムです。

    実行すると

	発音します
	    ↓
	|> ボタン (START) でミュート ON
	    ↓
	|> ボタン (START) でミュート OFF
	    ↓
	|> ボタン (START) で終了

    の処理を行います。
	
【プログラムの作成】

	> psymake tuto2

\sample\sound\tuto\tuto3.c

  割り込みのサンプルプログラムです。

  ・DMA 転送で

	sin 波		(割り込み起動チェック音)
	ピアノの音	(割り込みをかける波形)

    の波形を送ります。

  ・割り込みアドレスはピアノの波形のちょうど中間に
    設定してあります。

  ・ボタンアサインは以下のようになっています。

	o |> ボタン (START) で発音

		→ ちょっとすると割り込みがかかり、
		   sin 波の音が発音されます。

	o ■ ボタン (SELECT) で消音

	o |> ボタン (START) と ■ ボタン (SELECT) を
	  同時に押すと終了

【プログラムの作成】

	> psymake tuto3

\sample\sound\tuto\tuto4.c

  ノイズ音源のサンプルプログラムです。

	Lup   ボタン (↑ボタン):	sin 波/音程 up
	Ldown ボタン (↓ボタン):	sin 波/音程 down
	Rup   ボタン (△ボタン):	ノイズ/音程 up
	Rdown ボタン (×ボタン):	ノイズ/音程 down

【プログラムの作成】

	> psymake tuto4


::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

\sample\sound\tuto\tuto5.c

  波形データの分割転送のサンプルプログラムです。

	サウンドバッファ内 2 つの領域に同じ波形データを 2048 バイト
	ずつ分割して転送し、発音を行います。

  ボタンアサインは以下のようになっています。

	o ■ ボタン (SELECT) で終了

【プログラムの作成】

	> psymake tuto5

\sample\sound\tuto\tuto6.c

  リバーブのサンプルプログラムです。

	リバーブのモードを変えて、ピアノを発音することができます。

  ボタンアサインは以下のようになっています。

	o |> ボタン (START) でピアノが発音
	o ■ ボタン (SELECT) でピアノが消音
	o |> ボタン (START) と ■ ボタン (SELECT) を
	  同時に押すと終了
	o ↑↓ボタンでリバーブモードの変更

【プログラムの作成】

	> psymake tuto6
