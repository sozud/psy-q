基本サウンドライブラリに含まれる SPU ストリーミングライブラリを使用し
たサンプルです。

────────────────────────────────────
tuto1.c:

7 チャンネル (ボイス数 14) の SPU ストリームをコントロールパッドの操作
によって再生/停止を行います。SPU ストリーミングの状態も画面に表示しま
す。

【パッドの操作】

	↑ボタン:	ボール数 増
	↓ボタン:	ボール数 減
	×ボタン:	SPU ストリーム全チャンネル同時開始
	○ボタン:	SPU ストリーム全チャンネル同時停止
	□ボタン:	全ボイス Key Off
	△ボタン:	プログラム終了

	START ボタンを押しながら:
		↑ボタン:	SPU ストリーム 0 チャンネル開始
		→ボタン:	SPU ストリーム 1 チャンネル開始
		↓ボタン:	SPU ストリーム 2 チャンネル開始
		←ボタン:	SPU ストリーム 3 チャンネル開始
		△ボタン:	SPU ストリーム 4 チャンネル開始
		○ボタン:	SPU ストリーム 5 チャンネル開始
		×ボタン:	SPU ストリーム 6 チャンネル開始

	SELECT ボタンを押しながら:
		↑ボタン:	SPU ストリーム 0 チャンネル停止
		→ボタン:	SPU ストリーム 1 チャンネル停止
		↓ボタン:	SPU ストリーム 2 チャンネル停止
		←ボタン:	SPU ストリーム 3 チャンネル停止
		△ボタン:	SPU ストリーム 4 チャンネル停止
		○ボタン:	SPU ストリーム 5 チャンネル停止
		×ボタン:	SPU ストリーム 6 チャンネル停止

【プログラムの作成】

	> psymake

【プログラムのロード・実行】
	
	> psymake load
	> run main

────────────────────────────────────
tuto2.c:

CD ストリーミングと SPU ストリーミングをミックスさせたサンプルです。

CD ストリーミングのコードは sample\cd\movie\tuto0.c をベースにしていま
す。

SPU ストリーミングは上記 tuto1.c とほぼ同じコードを使い、7 チャンネル 
(ボイス数 14) の SPU ストリームをコントロールパッドの操作によって再生/
停止を行います。

		↓ボタン:	SPU ストリーム 2 チャンネル開始
		←ボタン:	SPU ストリーム 3 チャンネル開始
		△ボタン:	SPU ストリーム 4 チャンネル開始
		○ボタン:	SPU ストリーム 5 チャンネル開始
