        ドロップシャドーサンプルプログラムの動かし方:

        PADn    .... 物体が動く
        PADl    .... 光源が動く
        PADR    .... 全体が動く

------------------------------------------------------------------

        How to execute the sample program to drop the shadow:

        PADn    .... The object moves.
        PADl    .... The light source moves.
        PADR    .... All moves.
