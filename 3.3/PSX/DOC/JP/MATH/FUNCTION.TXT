数学ライブラリ


pow	xのy乗

	形式	double pow(double x, double y)

	引数	x	数値
		y	べき乗

	解説	xのy乗を計算する。

	返り値  xのy乗（x**y）

	備考	エラー処理は次の通り。

		    条件                   返り値       エラー
		-------------------------------------------------
		x==0 && y>0                  0        
		x==0 && y<=0                 1        領域エラー
		x<0 && 「yが整数値でない」   0        領域エラー


	関連項目	exp()
		


exp	指数

	形式	double exp(double x)


	引数	x	浮動小数点値	

	解説	xの指数関数を求める。

	返り値  eのx乗（e**x）

	関連項目	pow(), log()



log	自然対数

	形式	double log(double x)

	引数	x	対数計算の対象となる値

	解説	xの自然対数関数を求める。

	返り値  xの対数（ ln(x) ）

	備考	xは0より大きいこと。それ以外の場合は範囲エラー。

		    条件     返り値       エラー
		-------------------------------------------------
		    x<0        0        領域エラー     
		    x==0       1        範囲エラー


	関連項目	exp(), log10()

	解説	xの基底10の対数関数を求める。

	返り値  xの基底10の対数（ log10(x) ）

	備考	xは0より大きいこと。それ以外の場合は範囲エラー。

		    条件     返り値       エラー
		-------------------------------------------------
		    x<0        0        領域エラー     
		    x==0       1        範囲エラー


	関連項目	log()



floor	xより大きくない最大の整数（低関数）

	形式	double floor(double x)

	引数	x	浮動小数点値	

	解説	xより大きくない最大の整数（double型）を求める。

	返り値  xより大きくない最大の整数（double型）

	関連項目	ceil()



ceil	xより小さくない最小の整数（天井関数）

	形式	double ceil(double x)

	引数	x	浮動小数点値	

	解説	xより小さくない最小の整数（double型）を求める。

	返り値  xより小さくない最小の整数（double型）

	関連項目	floor()



fmod	x/yの浮動小数点数余り

	形式	double fmod(double x, double y)

	引数	x	浮動小数点値	
		y	浮動小数点値	

	解説	x/yの浮動小数点数余りを求める。

	返り値  x/yの浮動小数点数余剰。

	備考	返り値の符号はxと同じ。
		yが0の場合は0を返す。



modf	整数部と小数部への分割

	形式	double modf(double x, double *y)

	引数	x	浮動小数点値	
		y	整数部を格納するためのバッファへのポインタ

	解説	xを整数部と小数部への分割する。
		整数部はyに格納し、小数部が返り値となす。

	返り値  xの小数部。

	備考	整数部、小数部ともに符号はxと同じ。

asin	逆正弦

	形式	double asin(double x)

	引数	x	逆正接を計算する値。範囲は [-1, 1]。

	解説	xの逆正弦関数を求める。

	返り値  xの逆正弦関数。範囲は [-pai/2, pai/2]
		エラー処理は次の通り。		

		    条件           返り値       エラー
		-------------------------------------------------
		    fabs(x)>1        0        領域エラー     


	備考	[]は閉領域を示す。

	関連項目	sin(), acos(), atan()

	解説	xの逆余弦関数を求める。

	返り値  xの逆余弦関数。範囲は [0, pai]
		エラー処理は次の通り。		

		    条件           返り値       エラー
		-------------------------------------------------
		    fabs(x)>1        0        領域エラー     


	備考	[]は閉領域を示す。

	関連項目	cos(), asin(), atan()

	解説	xの逆正接関数を求める。

	返り値  xの逆正接関数。範囲は [-pai/2, pai/2]

	備考	[]は閉領域を示す。

atan2	逆正接

	形式	double atan2(double x, double y)

	引数	x	浮動小数点値		
		y	浮動小数点値	

	解説	x/yについて逆正接関数を求める。

	返り値  x/yの逆正接関数。範囲は [-pai, pai]
		エラー処理は次の通り。		

		    条件           返り値       エラー
		-------------------------------------------------
		    x==0 && y==0     0        領域エラー     		


	備考	[]は閉領域を示す。

	関連項目	atan()

hypot	複素数絶対値

	形式	double hypot(double x, double y)

	引数	x	浮動小数点値
		y	浮動小数点値

	解説	複素数x+iyの絶対値を求める。

	返り値  x**2とy**2の和の平方根。



ldexp	仮数と指数から実数を計算（x×2**n）

	形式	double ldexp(double x, long n)

	引数	x	浮動小数点値
		n	整数の指数

	解説	仮数と指数から実数を計算する。

	返り値  x×2**nの値



frexp	正規化小数部と２のべき乗部への分解

	形式	double frexp(double x, int *n)

	引数	x	浮動小数点値
		n	２のべき乗部を格納するバッファへのポインタ

	解説	xを[1/2,1)に正規化された小数部と２のべき乗部に分解する。
		小数部は返り値となり、べき乗部はnに格納される。

	返り値  正規化小数部。[1/2, 1)

	備考	[]は閉区間、()は開区間を示す。

	形式	fabs(double x)

	引数	x	浮動小数点値	

	解説	絶対値を求める。

	返り値  xの絶対値。

	備考	マクロである。libmath.hで定義されている。
