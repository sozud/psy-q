対戦ケーブルライブラリ

関数詳細

-----------------------------------------------------------------------------
AddCOMB		対戦ケーブルドライバを初期化する。

	形式	void AddCOMB( void )

	引数	なし
	
	解説	対戦ケーブルドライバを初期化する。
	
-----------------------------------------------------------------------------
DelCOMB		対戦ケーブルドライバをカーネルから削除する。

	形式	void DelCOMB( void )

	引数	なし

	解説	対戦ケーブルドライバをカーネルから削除する。



-----------------------------------------------------------------------------
ChangeClearSIO	ドライバ内で拡張 SIO からの割り	込みをクリアする。

	形式	void ChangeClearSIO( val )

	引数	val	割り込み要因クリアフラグ

	解説	val に非 0を指定するとドライバ内で拡張 SIO からの割り
		込みをクリアします。他の拡張 SIO ドライバと共存する場
		合にのみ使用します。
